JumpBlocker / 开屏弃

A cleaner launch path—no VPN, no traffic filtering.

Skip splash countdowns and shake-to-open surprises by opening apps from Home Screen Web Clips or the in-app launcher via official deep links.

English

JumpBlocker helps you add Home Screen shortcuts that open supported apps through URL schemes, often bypassing the splash-heavy cold start triggered by the original app icon. Profiles are generated on your device when you export them.

Core Features

  • Web Clip export - Export a configuration profile with Web Clips for 25+ popular apps (all or selected)
  • Quick Open grid - One-tap deep-link launcher inside the app
  • Optional widget - Medium Home Screen widget for curated apps
  • Install guide - Step-by-step help for installing profiles in Settings
  • Lightweight - No VPN, DNS, or network extension; works alongside any VPN you already use

How to Install Web Clips

  • Open JumpBlocker → Web Clips → Add all apps or pick individual apps → Export
  • Open the .mobileconfig file → Settings → Install Profile (unsigned warning is normal)
  • Use the new Home Screen icons instead of the original app icons for best results

What JumpBlocker Is Not

  • Not a system-wide ad blocker (no in-feed, video, or network ad blocking)
  • Does not intercept or read your network traffic
  • Does not install MDM, VPN, or remote management beyond Web Clip shortcuts you choose
  • Results vary by app and iOS version; third-party names describe compatibility only

FAQ

  • Why “Unsigned”? Profiles are generated locally and are not Apple-notarized; iOS shows a standard warning before install.
  • App won’t open? The target app may be uninstalled or the URL scheme may have changed—try updating JumpBlocker when a new registry version is available.
  • Widget vs Web Clip? Web Clips on the Home Screen are the primary path; the widget is a supplementary shortcut.
中文（简体）

开屏弃帮助您通过主屏幕 Web Clip 或应用内快速打开，使用官方深度链接（URL Scheme）启动受支持的 App， 常能绕开点击「原图标」时的开屏与摇一摇跳转路径。描述文件仅在您导出时在设备本地生成。

核心功能

  • Web Clip 导出 - 一键导出含 25+ 常用 App 的描述文件（支持全选或自选）
  • 快速打开 - 应用内网格，一点即通过深度链接启动
  • 可选小组件 - 中号主屏幕小组件，方便常用 App
  • 安装指引 - 图文说明如何在「设置」中安装描述文件
  • 轻量无负担 - 无 VPN、DNS 或网络扩展，可与现有 VPN 共存

如何安装 Web Clip

  • 打开开屏弃 → Web Clip → 添加全部或单个 App → 导出
  • 打开 .mobileconfig 文件 → 设置 → 安装描述文件（「未签名」提示属正常现象）
  • 请使用主屏幕新图标打开 App，而非原来的 App 图标，效果更佳

开屏弃不是什么

  • 不是全网广告拦截器（不屏蔽信息流、视频中插或网络广告）
  • 不拦截、不读取您的网络流量
  • 除您主动安装的 Web Clip 外，不安装 MDM、VPN 或远程管理
  • 效果因 App 与系统版本而异；第三方名称仅作兼容性说明

常见问题

  • 为何提示「未签名」？ 描述文件在本地生成，未经 Apple 公证；安装前 iOS 会显示常规警告。
  • 打不开目标 App？ 可能未安装该 App，或 URL Scheme 已变更—请尝试更新开屏弃以获取最新方案列表。
  • 小组件和 Web Clip 区别？ 主屏幕 Web Clip 是主要路径；小组件为辅助快捷方式。
