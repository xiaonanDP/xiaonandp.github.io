PiPmaster / 画中画大师

Record screen + facecam or teleprompter in one flow.

Create tutorials, demos, and game commentary faster with floating camera PiP and teleprompter PiP.

English

PiPmaster helps creators record polished videos by combining iOS screen recording with a floating picture-in-picture layer.

Core Features

  • Camera PiP - Floating front/rear camera overlay for facecam-style recordings
  • Teleprompter PiP - Script overlay with font size and high-contrast options
  • Speech Follow - Optional voice-guided highlighting and scrolling
  • Image/Video PiP - Import media and keep references visible while recording
  • Local-First Workflow - Media stays on your device unless you share it

How to Start

  • Open PiPmaster and choose Camera, Text, Images, or Video
  • Set your source and tap Start PiP
  • Start iOS system screen recording, then switch to your target app

Known Issues

  • Behavior may vary by iOS version and device performance.
  • Text-only PiP currently does not export as a standalone video file.
中文（简体）

画中画大师帮助你把 iOS 录屏与悬浮画中画能力结合起来，更高效完成教程、演示和游戏解说视频。

核心功能

  • 画中画相机 - 前后摄小窗出镜，录屏讲解更自然
  • 悬浮提词 - 支持字号和高对比度设置，阅读更清晰
  • 跟读提词 - 可选语音跟随高亮与滚动
  • 图片/视频画中画 - 录制时保留参考内容
  • 本地优先 - 内容默认保存在本机，按需分享

快速开始

  • 打开画中画大师，选择相机、提词、图片或视频
  • 完成设置后点击 Start PiP
  • 启动 iOS 系统录屏，再切换到目标 App 开始录制

已知问题

  • 不同 iOS 版本和设备性能下，表现可能存在差异。
  • 纯提词画中画目前不导出独立视频文件。
